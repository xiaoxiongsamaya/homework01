## 如何使用

### 1. 获取 DeepSeek API Key
- 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
- 注册账号并登录
- 进入 [API Keys 页面](https://platform.deepseek.com/api_keys)
- 点击“创建 API Key”，复制保存

### 2. 运行项目
- 直接用浏览器打开 `index.html`
- 在页面中粘贴你的 API Key
- 输入主题，点击生成即可
